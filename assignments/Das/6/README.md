# Abstract

Electronic Theses and Dissertations (ETDs) are valuable scholarly resources, but they remain difficult to process automatically because of challenges such as inconsistent metadata, OCR errors, citation variability, and complex document layouts. This dissertation introduces **ETDSuite**, a toolkit that leverages Natural Language Processing (NLP) and Computer Vision (CV) techniques to mine and enrich scholarly ETD data.

The research proposes several AI-driven frameworks including:
- **AutoMeta** for automatic metadata extraction  
- **MetaEnhance** for metadata quality improvement  
- **ETDPC** for ETD page classification and segmentation  
- **LMParsCit** for citation parsing across multiple citation styles  

The dissertation also contributes benchmark datasets such as **ETD500**, **ETDQual500**, and **ETDCite** for evaluating ETD mining tasks. Experimental results demonstrate improved metadata extraction, segmentation accuracy, metadata quality enhancement, and robust citation parsing across multiple academic disciplines and citation styles. The overall goal is to transform static ETDs into intelligent, machine-readable, and searchable scholarly knowledge systems. :contentReference[oaicite:0]{index=0}

# Key Contributions

- Developed **AutoMeta** for extracting metadata from scanned and born-digital ETDs  
- Proposed **MetaEnhance** to improve incomplete, inconsistent, and incorrect metadata  
- Introduced **ETDPC**, a multimodal ETD page classification framework using text and vision features  
- Developed **LMParsCit** for parsing multi-style citations using language models  
- Created benchmark datasets including **ETD500**, **ETDQual500**, and **ETDCite**  
- Improved scholarly discovery and accessibility of ETD repositories using AI methods
